# 长期记忆评测标准

MEMORY.md 用于维护需要长期保留和持续更新的事项，不用于记录一般用户画像。

## 评分维度与权重

- 正确性 `correctness`：30%
- 完整性 `coverage`：20%
- 更新合理性 `update_logic`：20%
- 记忆边界 `memory_boundary`：15%
- 去重凝练 `conciseness`：10%
- 格式合规 `format`：5%

## 核心边界

- 允许记录：长期计划、跟踪事项、已决定的未来安排、娱乐进程、旅行事项、持续关注点、车机习惯。
- 禁止记录：独立用户画像、他人独立属性、敏感证件信息、一次性任务、瞬时行为情绪、假设及创作内容。
- 仅以 user 内容作为新事实来源，assistant 内容不能独立成为记忆。
- 未提及的旧记忆不得删除；明确冲突时新值覆盖旧值；同类事项应合并去重。
- 非空输出使用 `- 字段名：内容`，不得混入标题、解释、代码块或推理过程。
- 没有有效长期记忆时允许输出空字符串。
