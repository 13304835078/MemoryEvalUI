MemoryEvalUI 协作者快速说明
==========================

这是源码协作版，不包含打包产物、真实配置、上传文件、评测结果或 API token。

一、环境
-------

推荐 Python 3.11。

安装依赖：

```powershell
pip install -r requirements.txt
```

二、运行
-------

源码方式启动：

```powershell
streamlit run app.py
```

或使用命令行菜单：

```powershell
python run.py
```

三、配置
-------

复制示例配置：

```powershell
copy config\local_config.example.json config\local_config.json
```

然后填写真实 API 地址、token、judge_model。

四、主要代码位置
---------------

- `pages/`：Streamlit 页面
- `src/schema.py`：Case / EvalResult / EvalConfig 数据结构
- `src/loaders/`：Excel、JSONL、Markdown 数据加载
- `src/eval/`：Judge 调用、评测流程、统计逻辑
- `src/ui/`：配置、文件、复核、展示组件
- `prompts/judge/`：Judge prompt
- `rules/`：评分规则
- `scripts/`：命令行转换与评测工具
- `tests/`：回归测试

五、测试
-------

```powershell
python -m pytest -q
```

当前重点任务是 `user_md_update`。`day_memory`、`long_memory`、`summary` 仍是预留入口。
